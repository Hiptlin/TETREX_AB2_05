About the Project

This dashboard is part of an advanced monitoring system designed to continuously track system performance and activity. It leverages real-time data such as CPU, memory, disk usage, file events, and network activity to detect unusual behavior. Using anomaly detection techniques, the system can issue early warnings if potential ransomware activity is detected.

The project utilizes a REST API and WebSocket connection (hosted on 127.0.0.1:8001) for live data updates and alert notifications. Explore the tabs above to view live tracking data or review detailed system information along with ransomware simulation tools.
